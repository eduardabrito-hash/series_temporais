# Séries Temporais

## Descrição

Este repositório reúne os materiais desenvolvidos na disciplina de **Séries Temporais** do Curso de Bacharelado em Estatística da **Universidade Estadual da Paraíba (UEPB)**.

Ao longo da disciplina são desenvolvidas atividades práticas, listas de exercícios, projetos, relatórios e apresentações, utilizando a linguagem **R** e o **Quarto** para análise, modelagem e previsão de séries temporais.

## Organização

### dados/

Contém as bases de dados utilizadas em cada atividade, lista de exercícios e projeto desenvolvido durante a disciplina.

### scripts/

Reúne os códigos em **R**, organizados por atividade e projeto, incluindo etapas de:

- Importação dos dados;
- Pré-processamento;
- Análise exploratória;
- Modelagem;
- Validação;
- Previsão.

### relatorios/

Contém os relatórios desenvolvidos em **Quarto (.qmd)**, bem como os arquivos gerados durante sua elaboração.

### apresentacoes/

Contém as apresentações utilizadas durante os seminários e atividades da disciplina.

## Como utilizar

1. Clone este repositório para o seu computador.
2. Abra o projeto no **RStudio**.
3. Navegue até a pasta correspondente à atividade ou projeto desejado.
4. Execute os scripts em **R**.
5. Para gerar os relatórios, renderize os arquivos **.qmd** utilizando o **Quarto**.
